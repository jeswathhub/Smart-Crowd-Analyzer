# Smart Crowd Density & Risk Zone Analyzer

This project detects people in live video streams using YOLOv8 and analyzes crowd density to identify high-risk zones.

## Features
- Real-time person detection
- Crowd density estimation
- Heatmap visualization
- Risk zone alerts
- Supports CCTV / phone videos
